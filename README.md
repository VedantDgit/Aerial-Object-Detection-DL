# Aerial Object Classification & Detection

A deep learning-based system to classify and detect aerial objects as **Bird 🐦** or **Drone 🚁** using CNN, Transfer Learning, and YOLOv8.

---

##  Project Overview

This project builds an intelligent system that can:

- Classify aerial images into **Bird** or **Drone**
- Detect and localize objects using **YOLOv8**
- Provide an interactive interface using **Streamlit**

##  Key Features

- 🔹 Binary Image Classification (Bird vs Drone)
- 🔹 Object Detection with Bounding Boxes (YOLOv8)
- 🔹 Transfer Learning (MobileNetV2, ResNet50)
- 🔹 Model Comparison & Evaluation
- 🔹 Clean and Interactive UI (Streamlit)

---

##  Models Used

| Model | Type | Description |
|------|------|------------|
| Custom CNN | Deep Learning | Baseline model |
| MobileNetV2 | Transfer Learning | Best performing model |
| ResNet50 | Transfer Learning | Deep architecture comparison |
| YOLOv8 | Object Detection | Real-time detection |

---

##  Results

- MobileNetV2 achieved ~97% accuracy  
- YOLOv8 successfully detects objects with bounding boxes  
- Minimal misclassification and strong generalization
